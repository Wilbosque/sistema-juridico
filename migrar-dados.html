Migração de Dados

Envia os processos, prazos e suspensões antigos (que ficavam só no navegador) para o banco de dados no Supabase. Rode uma vez por navegador que tinha dados.

1. Entrar

Use o mesmo e-mail e senha do sistema. Isso é necessário para gravar com segurança no banco.

2. Migrar

Opção A (recomendada): selecione o arquivo de backup .json que você exportou do sistema antigo.

Opção B: migrar os dados que ainda estão guardados neste navegador. Use só se você ainda não abriu a versão nova do sistema aqui.
